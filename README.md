# 神奇计算器

一个不会算数的计算器 -- 输入公式后自动跳转豆包 AI 帮你算。

## 它能做什么

在计算器界面上输入算式，按下 `=` 后：

1. 自动打开浏览器跳转到 [豆包](https://www.doubao.com/chat/)
2. 检测页面加载完成后自动粘贴公式并发送
3. 豆包帮你算出结果

## 使用方式

### 方式一：直接运行 exe（推荐）

下载 `神奇计算器.exe`，双击即可运行，无需安装 Python。

### 方式二：运行源码

```bash
pip install keyboard
python calculator.py
```

### 自行打包 exe

```bash
pip install pyinstaller keyboard
pyinstaller --onefile --noconsole --name "神奇计算器" calculator.py
```

生成的 exe 在 `dist/神奇计算器.exe`。

## 键盘快捷键

| 按键 | 功能 |
|------|------|
| `0-9` / `.` | 输入数字 |
| `+` `-` `*` `/` | 运算符 |
| `(` `)` | 括号 |
| `Enter` / `=` | 发送到豆包计算 |
| `Backspace` | 退格删除 |
| `Esc` | 清除全部 |

## 工作原理

1. 用户在 tkinter 界面输入算式
2. 按下 `=` 后，公式复制到剪贴板，浏览器打开豆包
3. 通过轮询浏览器窗口标题检测页面是否加载完成
4. 页面就绪后，用 `keyboard` 库模拟 Ctrl+V 粘贴 + Enter 发送

## 系统要求

- Windows 10/11
- 需要一个浏览器（Edge / Chrome / Firefox 等）

## 依赖

- Python 3.10+（仅源码运行时需要）
- [keyboard](https://github.com/boppreh/keyboard) -- 键盘事件模拟
